# Community Positioning

ICS Calendar Viewer overlaps with calendar-related Obsidian plugins only at the file format boundary.

- ICS imports events from calendar/ICS URLs into Daily Notes.
- Full Calendar provides a full calendar interface and supports read-only remote ICS/CalDAV calendars.
- MemoChron focuses on calendar integration and event notes.
- iCal generates calendar output from vault tasks.

This plugin takes a narrower role: inspect a local `.ics` file in place without sync, import, write-back, RSVP, notifications, or event note creation.
